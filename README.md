# FIFA World Cup 2026 - Group G Analysis

## Overview

This project analyzes the recent performance of the national teams in Group G of the FIFA World Cup 2026 using data-driven methods.

The analysis focuses on:

- Recent team form
- Offensive and defensive performance
- Goal production and concession trends
- Power rankings
- Comparative metrics
- Tactical indicators

The project uses Python for data collection, cleaning, analysis, and visualization, while Tableau is used for interactive dashboards.

---

## Teams Analyzed

- Belgium
- Egypt
- Iran
- New Zealand
